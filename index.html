Project Description

The dataset used in the project is IAM Handwriting Dataset.The database contains forms of unconstrained handwritten text, which were scanned at a resolution 300 dpi and saved as PNG images with 256 gray levels. IAM Handwriting Database 3.0 is structured follows: The database contains forms of unconstrained handwritten text, which were scanned at a resolution 300 dpi and saved as PNG images with 256 gray levels. IAM Handwriting Database 3.0 is structured follows:

  • 657 writers contributed samples of their handwriting.
  • 1,539 pages of scanned text.
  • 5,685 isolated and labeled sentences.
  • 13,353 isolated and labeled text lines.
  • 115,320 isolated and labeled words.

To address the issue of class imbalance in our dataset, we employed a method of superimposing strike patterns over non-strike-through text. This approach allowed us to create a balanced dataset comprising both strike-through and non-strike-through text examples, ensuring that the model had equal representation of both classes during training. Using this enriched and diverse dataset, we trained a robust classifier model, EfficientNet80, which is well-suited for high-accuracy image classification tasks. The model demonstrated exceptional capability in accurately distinguishing between strike-through and non-strike-through text, paving the way for its effective application in improving OCR systems by filtering out irrelevant strike-through text during document digitization processes.
